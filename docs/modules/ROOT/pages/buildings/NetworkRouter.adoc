= Network Router
:description: The Network Router allows you to compartmentalize your FIC-SIT networks for easy modification.

|Building Type

|Max Network Connections

The Network Router allows you to compartmentalize your FIC-SIT networks for easy modification,
while still allowing cross-network communication.

Filters can also be defined based on message port and address.
Filters are set to Blacklist by default.

=== Functions

==== `setPortWhitelist(bool bInWhitelist)`

Sets if the port-filter should run in Whitelist-Mode or Blacklist-Mode.

Parameters::
+
[cols="1,1,4a"]

|Name |Type |Description

|bInWhitelist
|bool
|true when the filter should run in Whitelist-Mode
|===
